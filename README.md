# Программа для работы с базой данных products
В фунционал программы входит работа с базой данных products, получение новых таблиц и статистических данных, основанных на данной таблице
## Оглавление
- Как установить и запустить
- Как использовать проект
## Как установить и запустить
1. Выполните команду 'git clone https://github.com/domnichds/products-mcko/' в терминале
2. Запустите необходимы файлы с расширением .py
## Как использовать проект
Вы можете использовать проект для анализа базы данных products.csv
- При открытии файла task_1.py создается таблица с новым столбцом total, а также открывается окно, в котором выводится сумма выручки
- При открытии файла task_2.py открывается окно с информацией о самом дорогом товаре в категории, стоящей первой по алфавиту
- При открытии файла task_3.py открывается окно, в котором пользователью предложено ввести название категории. После получения данных программа выводит информацию о самом редкопродоваемом продукте в этой категории. Для завершения работы необходимо подать в программу слово 'молоко'
- При открытии файла task_4.py создается таблица с новым столбцом promocode. В нем указан промокод на скидку для каждого продукта. Код формируется по следующему правилу: 2 первые буквы названия + день поступления + 2 предпоследних буквы названия в обратном порядке + месяц поступления в обратном порядке
- При открытии файла task_5.py выводится список из категорий, товары которых продавались реже всего
